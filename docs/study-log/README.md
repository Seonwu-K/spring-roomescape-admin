# 학습법 정리
## v1
- 문제로부터 학습 포인트 결정
- 학습 포인트에 대해서 이해할 때까지 AI와 자료를 기반으로 학습(공식문서, 책, 코치의 조언)
- 이해가 되면 다시 문제 해결 과정으로 돌아가 미션을 진행
- 문제 : 학습을 하다 보면, 추가로 학습할 것이 계속 의식돼서 추가 학습을 진행하다 시간 배분에 실패하고 처음에 무엇을 위해 학습을 시작했는지 잊는 경우 발생
## v2
- 학습 시작하기 전에 학습할 주제를 미리 정해놓고 해당 주제에 대해서만 집중하여 학습
- 회고 : 무엇에 대해 학습할 것인지 미리 정해두고, 다음에 다시 학습하게 될 것임을 상기하며 더 깊게 들어가지 않고 처음 목표로 한 부분만 학습하고 마무리 할 수 있었다. 
- 문제 : 내가 학습하고자 하는 것이 처음부터 명확하고, ‘무엇을 얼만큼 왜 학습하는가’가 스스로 납득이 되기 때문에 가능했지만 그렇지 않으면 학습의 기준이 흔들릴 수 있을 것 같다.

## v3
- Metalearning Map : 학습 시작 전, 주제의 전체적인 구조를 먼저 확인
- 학습 범위 정의 + 학습하지 않을 내용 분류
- 학습 시작 전 시간 제약 설정(미션 진행에 지장이 가지 않을 시간을 계산해서 설정하고 해당 시간 내에 학습하기를 목표로)
- 종료 기준 정의 : 학습한 내용을 덮고 말이나 그림으로 설명할 수 있는가, 코드로 검증할 수 있는가
- 이후 제대로 이해한게 맞는지 피드백(리뷰)을 통해 확인
